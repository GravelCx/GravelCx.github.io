第一阶段

第一周 Day02

一. 什么是IDE

​ IDE(开发环境): 开发代码的工具

​ 开发环境: 记事本 –> Notepad++(目前阶段使用) –> Eclipse –> Idea

二. 编码问题

  1. 问题: 用notepad++有利于熟悉Windows系统的cmd黑窗口,但是首次使用会有编码问题

  2. 原因: notepad++默认语言是UTF-8,Windows电脑默认编码格式是GBK

  3. 解决方法

    方法一(推荐): 在notepad++中点击设置 首选项 新建 编码下方选择第一个ANSI 即可

​ 方法二: 每次在cmd编译时,加上编码格式,此方法很繁琐

​ 如: javac -encoding UTF-8 Hello.java

三. 标识符

  1. 什么是标识符

    含义: 给类,接口,方法,变量等起名字使用的字符序列

  2. 标识符的命名规范

    组成成分:

    ​ 英文大小写 或 数字字符 或 $ 和 ~

  3. 注意事项: 

    * 不能以数字开头
    • 特殊符号只能用 _ 和 $
    • 不能是Java中的关键字
    • 区分大小写

五. 变量

  1. 含义:

​ 程序运行过程中有些值需要随着实际情况改变的值,变量就是用来记录程序中会变的数据的值

  1. 书写格式:

​ 数据类型 变量名 = 初始值

如: int i = 18;

六. 数据类型的划分

  1. 含义:

​ Java语言是强类型语言,每一种格式的数据都有明确的数据类型,在内存中为每种数据类型分配了不同的内存空间

  1. 基本数据类型

    Java中，一共有8种基本数据类型：

​ 整数型(4种)：int,short,long,byte

​ 浮点型(2种)：float,double

​ 字符类型(1种)：char

​ 表示真值的类型(1种)：boolean

  1. 引用数据类型:

​ 如: String(字符串),它实际上是一个char类型的数组

七. 变量使用的注意事项

  1. 变量要先初始化,才能使用
  2. 变量有作用域,同一作用域不能重名

九. 运算符

  1. 算术运算符: + - * / % ++ –

    扩展：

    经典面试题一：输出结果为？
int a = 8;
int b = (a++)+(++a)+(a*10);
System.out.println(b);
​    经典面试题二：输出结果为？
​    int  i =  0 ;   

​    i = ++i; 
​    System.out.println(i);

​    底层实现：i = (int)(i+1); i=i;



​    int i = 0;   

​    i = i++;  

​    System.out.println(i);  

​    底层实现：int temp = i; i = (int)(i+1); i = temp;

  1. 赋值运算符: = += -= *= /= %=

    扩展:

    ​ 经典面试题一：输出结果为？
    ​ int a,b;
    ​ a = b = 100;
    ​ System.out.println(a);
    ​ System.out.println(b);

​    经典面试题二：下面的两种写法结果分别是？
​    short s=1; 
​    s = s+1; 

​    short s=1; 
​    s += 1;



赋值规则：
复合赋值 E1 op= E2等价于简单赋值 E1 = (T)((E1) op     (E2))，其中T是E1的类型。

可以简单理解为: 先运算再赋值(结果会被强转为E1的类型)

  1. 关系运算符: == != > < >= <=

    注意:

    ​ 关系运算符的结果都是boolean类型,true或者false

    ​ 不要将==和=混淆;

    扩展：

    经典面试题一：输出结果为？

    int x = 10;

    int y = 10;

    boolean flag = (x == y);

    System.out.println(flag);

    flag = (x = y);

    System.out.println(flag);

经典面试题二：输出结果为？

boolean b1 = true;

boolean b2 = false;

boolean b3 = (b1 == b2);

System.out.println(b3);

boolean b4 = (b1 = b2);

System.out.println(b4);

4.逻辑运算符: & && | || ! ^

​ &和&&: 两侧同true时,结果为true

​ |和||: 两侧只要有一个为true,结果就是true

​ !: 取反

​ ^: 两侧相同为false,不同为true

​ 效率方面: &&和||效率更高, 如:true||false时,通过左边即可得到结果true, 右边不会被读取

十. 拓展知识

​ Scanner类,目前知道用于获取控制台输入即可

  • 使用需要导包 import java.util.Scanner;
  • 创建对象: Scanner scan = new Scanner(System.in);
  • 获取输入: int num = scan.nextInt(); //此处进行方法的调用
